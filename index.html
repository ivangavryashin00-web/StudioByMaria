Меня зовут Мари, и я создаю авторские игрушки ручной работы. Каждый персонаж — это маленькая история, придуманная специально для вас.

В моей мастерской живут Бабы-Яги с характером, добрые домовые, весёлые вороны, нежные куколки и народные обереги. Все шьются вручную из качественных тканей с любовью.

Принимаю индивидуальные заказы — создам любого персонажа по вашему описанию или рисунку. Ваша игрушка будет единственной в мире.

Красиво упаковываю в подарочную коробку и отправляю по России. Почта, СДЭК или самовывоз.

☁️Облачная синхронизация (Firebase)

Изменения сохраняются в облако — одинаково на любом устройстве и браузере.
Инструкция (1 раз, полностью бесплатно):
① console.firebase.google.com → войти через Google → Создать проект
② Слева: Realtime Database → Создать базу данных → Тестовый режим
③ Скопируйте URL базы — вида https://ИМЯ-default-rtdb.firebaseio.com
④ ⚙ Настройки проекта → Сервисные аккаунты → Секреты базы данных → Показать → скопируйте

Не настроено. Введите URL и Secret → нажмите «Подключить Firebase».
🌐Публикация сайта (один раз)

Шаг 1 — Настройте Firebase (блок выше) и нажмите «Сохранить в облако»
Шаг 2 — Скачайте сайт кнопкой ниже (ключи вшиваются автоматически)
Шаг 3 — Выложите файл на хостинг:
  → GitHub Pages (бесплатно навсегда):
    ① Создайте аккаунт → New repository → имя: masterskaya
    ② Загрузите файл, переименуйте его в index.html
    ③ Settings → Pages → Deploy from branch → main → Save
    ④ Сайт будет на: ВАШ-НИК.github.io/masterskaya

После публикации: все изменения через панель → «☁ Сохранить в облако» → клиенты видят сразу!

Файл уже содержит ваши Firebase-ключи — клиенты будут автоматически видеть актуальные данные.
